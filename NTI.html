Распознавание археологических объектов

LiDAR + CNN · Участие в конкурсе НТИ «Экспедиция. Data Science»

О проекте

Разработка системы искусственного интеллекта для обнаружения археологических объектов (курганов, фундаментов, дорог) по данным LiDAR и мультиспектральной съёмки. Проект выполняется в рамках технологического конкурса Up Great «Экспедиция. Data Science» при поддержке НТИ.

Основные вызовы: работа с зашумленными данными, распознавание объектов на глубине до 5 метров, высокая точность (требование >90%). В проекте применяются современные архитектуры CNN (U-Net, DeepLabV3+), методы аугментации геопространственных данных, а также ансамблирование моделей.

Научные этапы разработки

1. Анализ научной литературы

Обзор

Изучение современных методов археологической разведки с использованием ML. Сравнение подходов на основе LiDAR и мультиспектральных данных. Выделены ключевые архитектуры: U-Net, DeepLab, Mask R-CNN.

2. Сбор и подготовка данных

Датасет

Получены открытые наборы LiDAR (USGS, OpenTopography) и данные организаторов конкурса. Произведена привязка, фильтрация шумов, создание масок для объектов.

3. Геопространственная предобработка

GIS

Конвертация сырых облаков точек в растры высот (DSM/DTM), вычисление индексов (интенсивность, мультиспектральные каналы). Использование GDAL, PDAL.

4. Проектирование нейросетевой архитектуры

U-Net

Эксперименты с U-Net, LinkNet, DeepLabV3+. Адаптация для многоканальных входных данных (до 8 каналов). Внедрение механизмов внимания (CBAM).

5. Аугментация пространственных данных

Augment

Разработаны методы аугментации, сохраняющие геометрию: случайные повороты, сдвиги, масштабирование, добавление шума к высотам.

6. Обучение и настройка

Train

Обучение на GPU-кластере ИТМО. Использование фокальных функций потерь (Focal Tversky) для борьбы с дисбалансом классов. Метрики IoU, F1.

7. Оценка и кросс-валидация

Validation

Проведена кросс-валидация по различным географическим регионам. Анализ ошибок на редких классах (погребения, малые объекты).

8. Ансамблирование и бустинг

Ensemble

Объединение предсказаний нескольких моделей (U-Net + DeepLab) с усреднением и обучением мета-классификатора. Повышение IoU на 5%.

9. Участие в конкурсе НТИ

КОЗ №2

Участие в Конкурсе отдельных заданий №2 (поиск подповерхностных объектов). Получены предварительные результаты выше baseline.

10. Подготовка публикации

Paper

Статья о методах обработки мультиспектральных данных для археологии. Подана на конференцию "Математические методы распознавания образов" (ММРО-2026).

11. Оптимизация инференса

ONNX

Конвертация модели в ONNX, оптимизация под CPU/GPU, создание пайплайна для обработки больших растров (tile-based inference).

12. Развёртывание и документация

Deploy

Разработан веб-интерфейс для визуализации результатов, подготовлен отчёт для конкурса, выложен код на GitHub.
